Your habits will sync across all devices via Firebase (free). One-time 5-minute setup.

1
Go to console.firebase.google.com → Add project → name it → disable Analytics → Create project
2
Left sidebar: Build → Realtime Database → Create database → any region → Start in test mode → Enable
3
Copy the Database URL shown at the top of that page — looks like https://your-project-default-rtdb.firebaseio.com
4
Click ⚙ gear icon (top-left) → Project Settings → General → copy Web API Key
5
Paste both below, pick a personal sync key (like a password), and hit Connect
